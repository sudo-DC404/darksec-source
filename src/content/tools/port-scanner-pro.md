A powerful port scanning tool built with Python for network reconnaissance and security testing.

## Features

- Fast multi-threaded scanning
- Service version detection
- Banner grabbing
- Export results to JSON/CSV
- Customizable timeout and thread count

## Installation

```bash
pip install port-scanner-pro
```

## Usage

```bash
python port_scanner.py -t target.com -p 1-1000
```
